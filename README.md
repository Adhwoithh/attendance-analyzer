# 📊 Attendance Intelligence & Prediction System

## 🚀 Features

* Attendance dashboard with visualization
* Risk detection based on threshold
* Future attendance simulation
* Subject-wise analysis
* Recommendation system for improvement

## 🛠️ Tech Stack

* Python
* Streamlit
* Pandas
* Matplotlib

## ▶️ How to Run

```bash
pip install -r requirements.txt
streamlit run app.py
```

## 📸 Output

* Interactive dashboard
* Risk alerts
* Student insights

## 🎯 Use Case

Helps students monitor and improve attendance efficiently.
